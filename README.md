# Bank-Loan-Default-Analysis

## Project Overview
Analysis of 255,347 bank loan records to identify key factors contributing 
to loan defaults using Python for data preparation, SQL for analysis, 
and Power BI for visualization.

## Tools Used
- Python (Pandas, Matplotlib) — Data Cleaning & EDA
- SQL Server (SSMS) — Database & Queries
- Power BI — Dashboard

## SQL Analysis
- Overall Default Rate
- Default Rate by Employment Type
- Average Credit Score & Income by Default Status
- Default Rate by Loan Purpose
- Default Rate by Education Level

## Dashboard Pages
- **Page 1 — Overview:** KPIs, Default Rate by Employment & Loan Purpose
- **Page 2 — Risk Analysis:** Default Distribution, Income & Education Analysis

## Key Findings
- Overall default rate is **11.61%** (29,653 out of 255,347 loans)
- **Unemployed** borrowers have the highest default rate at **13%**
- **High School** educated borrowers default more at **12%**
- Defaulters have lower avg income **($71,844)** vs non-defaulters **($83,899)**
- **Business** loans have the highest default rate at **12%**
- **Home** loans have the lowest default rate at **10%**
